# Security Policy

We take security issues seriously. If you discover a vulnerability in this
plugin, please report it privately so we can address it before public
disclosure.

Report a security issue

- Preferred: open a private security advisory on GitHub or use the repository's security contact.
- Email: security@mejkers.com

Response process

1. We will acknowledge receipt within 48 hours.
2. We will triage and assess the report and provide an estimated timeline.
3. We will coordinate a fix and release a patched version.
4. Once a fix is released, we will publicly disclose the issue unless you
   request otherwise.

Please include as much detail as possible: affected versions, steps to
reproduce, screenshots and any logs.
